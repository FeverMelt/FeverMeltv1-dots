## Dependecies
In order for the dotfiles to work properly (sans settings specific to my pc), you will need :  
 - btop  
 - cava  
 - fastfetch
 - hyprland
 - hyprpaper
 - kitty
 - millennium (for Steam customization)
 - rofi
 - solaar (Logitech mouse settings)
 - swaync
 - waybar  

```bash
  yay -S btop cava fastfetch hyprland hyprpaper kitty millennium rofi solaar swaync waybar
```
